# Portifólio - Matheus Fernandes

Um portifólio moderno e interativo desenvolvido com Next.js e Three.js, apresentando projetos e oferecendo forma de contato com animações 3D.

**[Acesse Online](https://rotinom0.github.io/portifolio/) -  Portifólio online

## 🚀 Tecnologias

- **[Next.js 16](https://nextjs.org)** - Framework React com SSR e otimizações
- **[React 19](https://react.dev)** - Biblioteca para construção de interfaces
- **[Three.js](https://threejs.org)** - Gráficos 3D no navegador
- **[Tailwind CSS 4](https://tailwindcss.com)** - Estilização utilitária
- **[Lucide React](https://lucide.dev)** - Ícones SVG

## 💻 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/rotinoM0/portifolio.git
cd portifolio
```

2. Instale as dependências:
```bash
npm install
```

## 🏃 Como Executar

**Modo de desenvolvimento:**
```bash
npm run dev
```
Abra [http://localhost:3000](http://localhost:3000) no navegador.

**Build para produção:**
```bash
npm run build
npm start
```

```
src/
├── app/

```
